Penjelasan Proses Deteksi Menggunakan CNN di Website
Website ini menggunakan Machine Learning yaitu CNN sebagai sistem utama untuk mendeteksi penyakit tanaman kedelai. Dataset yang digunakan pada pembuatan sistem ini diambil dari situs datadryad.org dengan judul “Pictures of diseased soybean leaves by category captured in fields and with controlled backgrounds: Auburn Soybean Disease Image Dataset (ASDID)” oleh Bevers, dkk (2022). Jadi website ini menggunakan model CNN yang sudah dilatih sebelumnya sebagai sistem deteksi yang dapat mengenali jenis penyakit tanaman kedelai berdasarkan citra daun. Untuk mengetahui lebih lanjut terkait kode sistem pada website ini, Anda dapat melihatnya di (https://github.com/ffdhil/Soybean-Diseases-Detection-ML).

Berikut adalah proses yang terjadi ketika sistem di website ini melakukan deteksi penyakit tanaman kedelai:
  1. Pengguna mengunggah gambar daun kedelai yang akan dideteksi pada tempat yang sudah disediakan di halaman deteksi kemudian menekan tombol "Deteksi".
  2. Sistem akan mengubah ukuran gambar menjadi ukuran 256x256 piksel karena model CNN yang telah dilatih menggunakan gambar dengan ukuran tersebut.
  3. Setelah berhasil mengubah ukuran, sistem akan mengubah setiap piksel gambar menjadi array dengan function img_to_array sehingga hasilnya akan menjadi array numpy dengan bentuk (256,256,3).
  4. Setelah berhasil mengubah gambar menjadi array, array tersebut akan di expand dimensinya menggunakan function np.expand_dims agar sesuai dengan format input model CNN sehingga hasilnya menjadi (1,256,256,3).
  5. Setelah berhasil, selanjutnya akan diproses pada model CNN yang telah dibuat sebelumnya.
